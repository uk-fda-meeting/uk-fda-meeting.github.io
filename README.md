Supported by the London Mathematical Society (LMS) Scheme 3 grant, this initiative aims to create a permanent, collaborative space for regular interaction and long-term joint research within the UK’s Functional Data Analysis community[cite: 1]. 

Our inaugural meeting features a dynamic format combining traditional research talks with intensive, interactive working groups designed to tackle open challenges and spark new collaborative projects[cite: 1].
